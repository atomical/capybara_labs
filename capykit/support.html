Common Issues

  • If an overlay does not appear, confirm the feature is enabled in the main CapyKit dashboard.
  • If a visual effect seems missing, check whether Focus, full-screen app suppression, or Reduce Motion is active.
  • If charger, Dock, or notch effects do not trigger, verify you are testing on a Mac that actually exposes that surface or event.
  • If you report a bug, include your Mac model, macOS version, and which CapyKit widget was active.
