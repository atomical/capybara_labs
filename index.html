Common Issues

  • Pope's Ring: If the game seems slow, restart the app and close other apps.
  • Quietune: If audio seems missing, verify the device is not muted, the current output route is correct, and the selected track is available for local playback.
  • Quietune: If Restore Purchases does not unlock the app, verify you are signed into the same App Store account that bought the lifetime unlock, then try again.
  • For bug reports, include the app name, device model, iOS version, and any steps needed to reproduce the problem.
